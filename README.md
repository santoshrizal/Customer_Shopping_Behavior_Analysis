#Customer Shopping Behavior Analysis
📌 Project Overview
This project analyzes customer shopping behavior using transactional data to uncover purchasing patterns, customer preferences, spending habits, and subscription trends. The analysis combines Python for data cleaning and exploratory analysis, SQL for business queries, and Power BI for interactive dashboard creation.
The objective of this project is to generate actionable business insights that can support data-driven decision-making, improve customer retention, and optimize business performance.
________________________________________
📊 Dataset Information
The dataset contains customer transaction records with information related to demographics, purchases, subscriptions, discounts, reviews, and shopping preferences.
Dataset Summary
•	Total Records: 3,900
•	Total Columns: 18
•	Data Type: Customer shopping transactions
•	Source: Public retail/customer behavior dataset
Key Features
•	Customer demographics (Age, Gender, Location)
•	Product and category information
•	Purchase amount and frequency
•	Subscription status
•	Discounts and promotional codes
•	Customer ratings and reviews
•	Shipping preferences
•	Seasonal purchasing behavior
